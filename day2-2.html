ServiceNow Certification Exam

Certified System Admin (CSA)

The ServiceNow Certified System Administrator (CSA) exam establishes the core skills required to manage, configure, and implement the ServiceNow platform.

Exam Key Details:

  • Format: 60 Questions (Multiple Choice)
  • Duration: 90 Minutes
  • Registration: Via the Webassessor portal.
  • Prerequisite: ServiceNow Fundamentals course is highly recommended.

Candidates are tested on User Interface & Navigation, Collaboration, Database Administration, and Self-Service applications.
