HTML

HTML stands for HyperText Markup Language.

It is used to create the structure of web pages using elements like headings, paragraphs, and links.

CSS

CSS stands for Cascading Style Sheets.

It is used to style and design web pages, including colors, layouts, and fonts.

JavaScript

JavaScript is a programming language used in web development.

It helps to add interactivity like animations, form validation, and dynamic content.
